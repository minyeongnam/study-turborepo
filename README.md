# turborepo-test

Turborepo 적용 테스트 프로젝트

## Project Run script

| pnpm             | Description          |
| ---------------- | -------------------- |
| `pnpm install`   | node modules install |
| `pnpm dev`       | dev client           |
| `pnpm storybook` | dev storybook        |

</br>

## Storybook 설치

```
cd apps/client || cd apps/admin
pnpx sb init --skip-install
pnpm install --save-dev @storybook/cli
```
